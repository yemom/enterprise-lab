# Product Service API (Lab 2)

RESTful Product Catalogue API built with Spring Boot.

## Run Locally

```bash
mvn spring-boot:run
```

Default base URL:

- http://localhost:8082/api/v1/products

## API Docs

- Swagger UI: http://localhost:8082/swagger-ui.html
- OpenAPI JSON: http://localhost:8082/api-docs

## Test

```bash
mvn test
```
